# Oster Warehouse & Trade Website

## How to upload to GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html` and `style.css` to the repository.
3. Go to Settings → Pages.
4. Select:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. Add your custom domain: `osterwarehouseandtrade.com`.
7. In your domain DNS settings, add GitHub Pages records.

A Records:
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153

CNAME:
www → yourgithubusername.github.io
